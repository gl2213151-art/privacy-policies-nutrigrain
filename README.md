# Nutrigrain Website

Sitio web oficial de Nutrigrain creado para proporcionar información sobre
nuestros productos, servicios y políticas relacionadas con la privacidad
de datos.

## Descripción

Nutrigrain es un negocio dedicado a la elaboración de postres y productos
a base de cereales, ofreciendo opciones elaboradas con ingredientes
seleccionados y atención personalizada a nuestros clientes.

Este sitio permite a los usuarios conocer:

- Información del negocio.
- Productos disponibles.
- Política de Privacidad.
- Términos y Condiciones.
- Proceso de eliminación de datos.
- Información de contacto.

## Tecnologías utilizadas

Este proyecto utiliza tecnologías web estándar:

- HTML5
- CSS3
- JavaScript

No requiere servidores adicionales y puede ser alojado utilizando
GitHub Pages.

## Privacidad y protección de datos

Nutrigrain se compromete a tratar la información de sus usuarios de forma
responsable y transparente.

La política de privacidad describe:

- Qué información puede recopilarse.
- Cómo se utiliza.
- Cómo se protege.
- Cómo los usuarios pueden solicitar la eliminación de sus datos.

## Integraciones

por Nutrigrain, incluyendo WhatsApp Business para atención al cliente,
